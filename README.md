Kenji Manuel Lao - IT Student Portfolio <br>

A modern, responsive portfolio website for an IT student specializing in Game Development. Features a dark/light theme toggle, interactive Snake game, and an enhanced certificate viewer. <br>

Features <br>
 Dual Theme Support <br>
Dark mode by default with smooth toggle to light mode <br>
Theme preference saved in browser's local storage <br>
Seamless transitions between themes <br>

Interactive Snake Game <br>
Classic Snake game built with pure JavaScript and Canvas <br>
Retro Game Boy-inspired design <br>
Score tracking with local high score storage <br>
Mobile-friendly touch controls <br>
Keyboard support (Arrow keys or WASD) <br>
Pause/Resume functionality <br>
Dynamic speed increase as score progresses <br>

Enhanced Certificate Gallery <br>
Click any certificate to view in full-screen modal <br>

Modal displays: <br>
High-resolution certificate image <br>
Complete title and description <br>
Provider and issue date <br>
Skills and technologies covered <br>
Navigation arrows to browse through certificates <br>
Keyboard navigation support <br>
Filter certificates by category (Networking, TESDA, Computer Systems) <br>

Responsive Design <br>
Fully responsive on all devices <br>
Mobile-optimized navigation <br>
Touch-friendly controls for the Snake game <br>
Adaptive layouts for different screen sizes <br>

Modern UI/UX <br>
Clean, professional design <br>
Smooth animations and transitions <br>
Font Awesome icons integration <br>
Google Fonts typography (Poppins & Roboto Mono) <br>
Interactive hover effects <br>

Technologies Used <br>
HTML5 - Structure and Canvas API for Snake game <br>
CSS3 - Styling, animations, and responsive design <br>
JavaScript - Interactive features and game logic <br>
Font Awesome - Icons and visual elements <br>
Google Fonts - Custom typography <br>

LocalStorage API - Theme preference and high score storage
