Kenji Manuel Lao - IT Student Portfolio

A modern, responsive portfolio website for an IT student specializing in Game Development. Features a dark/light theme toggle, interactive Snake game, and an enhanced certificate viewer.

✨ Features
🌓 Dual Theme Support
Dark mode by default with smooth toggle to light mode
Theme preference saved in browser's local storage
Seamless transitions between themes

🎮 Interactive Snake Game
Classic Snake game built with pure JavaScript and Canvas
Retro Game Boy-inspired design
Score tracking with local high score storage
Mobile-friendly touch controls
Keyboard support (Arrow keys or WASD)
Pause/Resume functionality
Dynamic speed increase as score progresses

📜 Enhanced Certificate Gallery
Click any certificate to view in full-screen modal

Modal displays:
High-resolution certificate image
Complete title and description
Provider and issue date
Skills and technologies covered
Navigation arrows to browse through certificates
Keyboard navigation support
Filter certificates by category (Networking, TESDA, Computer Systems)

📱 Responsive Design
Fully responsive on all devices
Mobile-optimized navigation
Touch-friendly controls for the Snake game
Adaptive layouts for different screen sizes

🎨 Modern UI/UX
Clean, professional design
Smooth animations and transitions
Font Awesome icons integration
Google Fonts typography (Poppins & Roboto Mono)
Interactive hover effects

🛠️ Technologies Used
HTML5 - Structure and Canvas API for Snake game
CSS3 - Styling, animations, and responsive design
JavaScript - Interactive features and game logic
Font Awesome - Icons and visual elements
Google Fonts - Custom typography

LocalStorage API - Theme preference and high score storage
